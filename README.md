# 조상일커피 - 프리미엄 랜딩페이지

## 📋 프로젝트 개요
조상일커피의 B2B 원두 납품 서비스를 위한 모바일 최적화 랜딩페이지입니다.  
PASONA 구조, 고객 니즈 분석, 경쟁력 있는 USP를 강조하여 제작되었습니다.

## 🧱 주요 기능
1. **​PASONA 프레임워크 적용​**  
   - Problem → Affinity → Solution → Offer → Narrow Down → Action
2. **​모바일 최적화​**  
   - 반응형 디자인 (320px ~ 1920px)
  . **​트렌디한 디자인​**  
   - 그라데이션, 블러 효과, 마이크로 인터랙션
4. **​TALLY 폼 연동​**  
   - 모든 CTA 버튼에 상담 신청 폼 연결
5. **​데이터 기반 신뢰 구축​**  
   - 고객 후기, 수치적 증거, 재구매율 표기
